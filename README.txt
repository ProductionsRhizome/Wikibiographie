WikiBiographie offre la gestion de biographies et la synchronisation du contenu avec Wikipédia en français.

== Description ==
Créez et gérez des biographies de personnalités. Affichez les informations provenant de Wikipédia en français et apportez vos propres changements au besoin.

== Installation ==
Déposer le dossier du plugin "wikibiographie" dans le dossier des plugins du site WordPress.
